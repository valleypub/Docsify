### 标识符命名规则
- 标识符用于给 变量、类、方法命名的符号
- 规则：
	- 1. 可以由字母、数字、下划线、美元符组成
	- 2. 数字不能打头
	- 3. 不能包含空格、@、#等其他特殊字符，只能包含美元夫（$）一种
	- 4. 不能是java的关键字、保留字。但可以包含关键字、保留字

### 关键字
